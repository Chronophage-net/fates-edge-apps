# Fate’s Edge – Plain‑English Licensing Advisory

Hi! I’m Nick (Chronophage), the creator of Fate’s Edge.  
This document explains in simple terms what you can and cannot do with the
code and game content in this repository.

## What’s free to use?

### 1. The software (the code that runs the Virtual Tabletop)
Licensed under the **MIT License** – you can:
- Run it for any purpose, personal or commercial.
- Modify the code, create your own fork, contribute improvements.
- Distribute the software as long as you keep the original copyright notice.
- Essentially, treat it like any other MIT‑licensed open‑source project.

### 2. The System Reference Document (SRD)
The SRD contains the bare‑bones mechanical rules of Fate’s Edge.  
It’s under **Creative Commons BY‑NC‑SA 4.0**, which means you can:
- Share and adapt it (make your own cheat sheets, explainer videos, etc.).
- Use it in your own non‑commercial projects.
- You **must** give me credit and share your adaptations under the same terms.

## What’s protected?

### 3. Everything else – the *Proprietary Content*
This is the heart of the game: the world of Acasia, the Curse, all the
characters, factions, spells, maps, art, and any data file not explicitly
labelled as SRD.  
**All Rights Reserved.**

**You may:**
- Use this content to play Fate’s Edge for free, either on the live server
  or on your own private instance of the software.

**You may NOT:**
- Sell it, include it in a paid product, or use it to run commercial games.
- Republish it on a website, wiki, or other platform.
- Create your own stories, adventures, or adaptations set in the world
  without my written permission.
- Distribute the data files separately from this software.

In short: the software is open so you can always have a table to play at.
The world belongs to me, and I’m happy to let you explore it — just don’t
take it and sell it or build a different game on top of it without asking.
